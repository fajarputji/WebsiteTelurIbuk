CARA PAKAI — WEBSITE TELUR IBUK
=================================

ISI FOLDER INI:
- index.html           -> halaman website (jangan dipindah dari folder ini)
- style.css            -> semua pengaturan warna, font, dan tata letak
- images/               -> folder gambar yang dipakai website

GANTI GAMBAR DENGAN FOTO ASLI
------------------------------
1. Siapkan foto telur asin kamu (format .jpg atau .png, usahakan background rapi)
2. Beri nama file foto SAMA seperti file yang mau diganti:
   - images/hero-egg-placeholder.png   -> gambar besar di bagian atas (hero)
   - images/produk-egg-placeholder.png -> gambar di kartu produk
3. Salin (copy-paste) foto baru kamu ke folder images, timpa file yang lama.
   Selesai — tidak perlu ubah kode apa pun di index.html.

Kalau mau pakai nama file lain, edit index.html, cari teks:
   images/hero-egg-placeholder.png
   images/produk-egg-placeholder.png
lalu ganti dengan nama file foto kamu (tetap di dalam folder images/).

GANTI NOMOR WHATSAPP & ALAMAT
------------------------------
Buka index.html dengan text editor (Notepad, VS Code, dll), lalu:
- Cari teks: 6281234567890       -> ganti dengan nomor WA asli kamu (format 62xxxxxxxxxx)
- Cari teks: Jl. Contoh Alamat   -> ganti dengan alamat toko asli kamu
- Cari teks: q=Semarang          -> ganti jadi alamat kamu, contoh: q=Jl+Pandanaran+Semarang

CARA MELIHAT HASILNYA
------------------------------
Klik dua kali file index.html, otomatis terbuka di browser (Chrome/Firefox/Edge).

CARA PUBLISH ONLINE (gratis, opsional)
------------------------------
Upload seluruh folder ini (index.html, style.css, dan folder images) ke layanan
seperti Netlify, Vercel, atau GitHub Pages agar website bisa diakses publik
lewat link.
